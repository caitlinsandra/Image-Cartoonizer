# Image Cartoonizer

## Overview
This project implements an image processing pipeline that transforms input images into stylized cartoon representations.

## System Architecture
- Image preprocessing
- Edge detection
- Color quantization
- Stylization transformation
- Output refinement

## Tools Used
Python, OpenCV, NumPy

## Key Learnings
- Structured pipeline development
- Parameter tuning and iterative debugging
- Handling edge cases across diverse image types
